1) რა არის state React ში და რითი განსხვავდება იგი props გან?

1. State

State არის კომპონენტის შიდა მონაცემები, რომლებიც შეიძლება შეიცვალოს კომპონენტის მუშაობისას. ეს მონაცემები სრულად კონტროლდება კომპონენტის მიერ, და მისი ცვლილება ანახლებს კომპონენტის UI-ს.

მახასიათებლები:

შინაგანად მართვადი: state მხოლოდ კომპონენტში შეიძლება იყოს ცვლილებული.

მოხდება განახლება: როდესაც state იცვლება, React ავტომატურად განაახლებს კომპონენტს.

მარტივი მნიშვნელობები: state შეიძლება იყოს ნებისმიერი ტიპის მონაცემი: მორჩილი ობიექტი, მასივი, სრეკვენცია, ციფრი და სხვა.




2. Props

Props (property) არის მახასიათებლები, რომელიც კომპონენტებს გადაეცემა და არ შეიძლება შეიცვალოს იმ კომპონენტში, რომელსაც ისინი გადაეცემა. props არის ბატონიანი (immutable) - ანუ მისი მნიშვნელობა არ შეიძლება შეიცვალოს კომპონენტში.

მახასიათებლები:

გადაეცემა კომპონენტიდან კომპონენტამდე: props განსაზღვრავს იმ მონაცემებს, რომელიც კომპონენტი იღებს საკუთარ მშობლიურ (parent) კომპონენტიდან.

Immutable: props არ შეიძლება შეიცვალოს კომპონენტის შიგნით; ისინი მხოლოდ          

სტატიკური: თუ props შეიცვალა, შესაბამისი კომპონენტი ავტომატურად განახლდება.
