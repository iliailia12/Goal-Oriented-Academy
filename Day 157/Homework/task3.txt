კომენტარების გამოყენებით თავიდან ბოლომდე ახსენით API. 
 


 API ნიშნავს Application Programming Interface-ს
  ქართულად შეგვიძლია ვთარგმნოთ როგორც — "აპლიკაციების პროგრამული ინტერფეისი".

  მარტივად რომ ვთქვათ:
  API არის წესი, გზები და მეთოდები, რომელთა საშუალებითაც ერთი პროგრამა მეორეს ესაუბრება.

  მაგალითად, როცა ვებგვერდი სერვერს აგზავნის მოთხოვნას და პასუხს იღებს მონაცემების სახით — ეს API-ით ხდება.

  მოდით, წარმოვიდგინოთ, რომ გვინდა მომხმარებლების სია წამოვიღოთ ერთი API-დან.
  ამისთვის ვიყენებთ `fetch()` ფუნქციას, რომელიც ბრაუზერის ჩაშენებული ფუნქციაა.



  API არის შუამავალი პროგრამებს შორის.

ერთი პროგრამა გზავნის მოთხოვნას (request), მეორე აბრუნებს პასუხს (response).

პროგრამები ერთმანეთს არ "ხედავენ", ისინი ურთიერთობენ API-ს საშუალებით.
